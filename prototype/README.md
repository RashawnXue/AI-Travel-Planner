# AI 旅行规划师 - 产品原型图

## 📖 概述

这是 AI 旅行规划师产品的 HTML 原型图，严格按照 PRD v1.0.0 文档设计，包含所有核心功能页面。

## 🎯 原型图特点

- ✅ **完整覆盖**：包含 PRD 中定义的所有核心页面和功能
- ✅ **高保真设计**：严格遵循 PRD 中的 UI 规范和交互细节
- ✅ **响应式布局**：支持桌面端、平板端和移动端查看

## 🚀 使用方法

### 方法一：直接打开文件

1. 使用浏览器打开 `index.html`
2. 点击页面中的卡片导航到各个功能页面

### 方法二：使用本地服务器（推荐）

```bash
# 在 prototype 目录下启动简单的 HTTP 服务器

# Python 3
python -m http.server 8080

# Node.js (需要安装 http-server)
npx http-server -p 8080

# 然后在浏览器访问
# http://localhost:8080
```

## 📄 页面说明

### 1. 用户认证模块

#### 注册页 (`register.html`)
- **路径**: `/register`
- **功能**: 
  - 用户名、邮箱、密码输入
  - 实时表单验证
  - 密码强度校验
  - 密码可见性切换
- **样式**: 居中卡片布局，渐变背景

#### 登录页 (`login.html`)
- **路径**: `/login`
- **功能**: 
  - 邮箱密码登录
  - 密码可见性切换
  - 跳转到注册页
- **样式**: 与注册页保持一致

### 2. 行程规划模块

#### 行程列表页 (`home.html`)
- **路径**: `/`
- **功能**: 
  - 左侧行程列表（可手动折叠/展开）
  - 点击行程卡片在右侧显示详情
  - 边栏不会自动收起，保持左右分栏展示
  - 顶部导航栏
  - 用户头像下拉菜单
  - 集成行程详情展示（包含4个Tab）
- **样式**: 左右分栏布局，左侧320px固定宽度

#### 创建行程页 (`create-plan.html`)
- **路径**: `/create`
- **功能**: 
  - 语音/文字输入切换
  - 语音录制动画效果
  - 可选补充信息折叠面板
  - 旅行偏好多选
  - 生成行程按钮
- **样式**: 居中卡片，最大宽度 800px

#### 行程详情页 (`plan-detail.html`)
- **路径**: `/plan/:id`
- **功能**: 
  - 行程概览区（目的地、天数、预算等）
  - Tab 导航（每日行程、住宿、交通、预算）
  - 每日行程时间轴展示
  - 活动导航功能
  - 费用预算可视化
  - 支出记录表格
- **样式**: 全宽布局，内容区最大宽度 1200px

### 3. 费用管理模块

#### 添加支出记录 (`add-expense.html`)
- **路径**: 弹窗组件
- **功能**: 
  - 语音/手动输入切换
  - 支出分类选择
  - 金额、描述、日期时间输入
  - 表单验证
- **样式**: 居中模态框，宽度 500px

## 🎨 设计规范

### 颜色规范

- **主色**: `#667eea` (紫蓝色)
- **悬停色**: `#5568d3`
- **渐变背景**: `linear-gradient(135deg, #667eea 0%, #764ba2 100%)`
- **成功色**: `#52c41a`
- **警告色**: `#fa8c16`
- **错误色**: `#ff4d4f`
- **文字主色**: `#1a1a1a`
- **文字次色**: `#666666`
- **边框色**: `#e8e8e8`

### 字体规范

- **字体**: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, sans-serif
- **标题大**: 32px / font-weight: 700
- **标题中**: 24px / font-weight: 600
- **标题小**: 18px / font-weight: 600
- **正文**: 14px / line-height: 1.6
- **辅助文字**: 12px / color: #999

### 圆角规范

- **卡片**: 12px
- **按钮**: 8px
- **输入框**: 8px
- **标签**: 4px
- **头像**: 50% (圆形)

### 间距规范

- **页面边距**: 24px
- **卡片内边距**: 32px (桌面) / 20px (移动)
- **元素间距**: 16px / 24px
- **表单项间距**: 20px

## 📱 响应式断点

- **桌面端**: ≥ 1280px
- **平板端**: 768px - 1279px
- **移动端**: < 768px

### 移动端适配

- 导航栏隐藏文字链接
- 左侧边栏默认折叠
- 表单卡片宽度改为 90%
- 网格布局改为单列
- Logo 尺寸缩小
